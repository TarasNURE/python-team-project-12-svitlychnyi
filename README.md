# Практичне заняття 6: Основи командної розробки Python-проєктів

Цей репозиторій містить виконання двох завдань у межах практичної роботи з дисципліни «Програмування на Python».

# 1. Базове завдання: «Консольний аналізатор даних»
Проєкт реалізує зчитування числових даних із файлу, їх обробку та обчислення базової статистики (середнє значення, мінімум, максимум, стандартне відхилення).

Структура:
    main.py — головний модуль інтеграції.
    
    io_utils.py — модуль вводу/виводу даних.
    
    analysis.py — модуль статистичного аналізу.
    
Як запустити:
  Bash
  
  python main.py
  
# 2. Індивідуальне завдання (Варіант 12): «Аналізатор лог-файлів»
Проєкт призначений для автоматизованого аналізу системних логів. Програма підраховує загальну кількість записів, кількість критичних помилок (ERROR) та попереджень (WARNING).

Структура:

main.py — головний скрипт для аналізу логів.

io_utils.py — функція зчитування лог-файлів.

analysis.py — функція обробки текстових записів.

Як запустити:

Bash
python main_logs.py
(Примітка: для роботи програми у папці має бути файл logs.txt)
