## 1. Базове завдання: «Консольний аналізатор даних»

Проста програма, яка зчитує числа з файлу, обробляє їх і видає базову статистику.

**Структура проєкту:**
* `main.py` — головний файл, який усе запускає.
* `io_utils.py` — відповідає за читання та запис даних.
* `analysis.py` — тут живе вся логіка статистичного аналізу.

**Як запустити:**
```bash
python main.py
2. Індивідуальне завдання (Варіант 12): «Аналізатор лог-файлів»
Цей скрипт автоматично аналізує системні логи. Він підраховує загальну кількість записів, а також шукає критичні помилки (ERROR) та попередження (WARNING).

Структура проєкту:

main_logs.py — головний скрипт для запуску аналізатора.

io_utils.py — модуль для зчитування лог-файлів.

analysis.py — модуль для пошуку та аналізу потрібних рядків.

Як запустити:

Bash
python main_logs.py
Важливо: щоб усе працювало, у кореневій папці обов'язково має лежати файл logs.txt.
